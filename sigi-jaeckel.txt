    1. What is Semantic HTML? 
    HTML that gives meaning to our content
    2. What is HTML used for? 
    It's used for the structure for our webiste - marking up our content
    3. What is an attribute and where do we put it? 
    Gives additional information to an element, put within the start tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
    Used for the most important heading of a page, should only be used once.
    5. Name two tags that have required attributes
    <img> and <a>
    6. What do we put in the head of our HTML document?
    The title and styling information
    7. What is an id? 
    An attribute that applies a name to an element that can be called for styling purposes
    8. What elements can I add an id to? 
    Any
    9. How many times can I use the same id on a page? 
    As many as you like, but should only be used once
    10. What is a class?
    An attribute that gives an element a class name used for styling purposes. Less specific than an id
    11. What elements can I add a class to? 
    Any
    12. How many times can I use the same class on a page? 
    As many as you like.
    13. How do I get my link to open in a new tab?
    target="_blank"
    14. What is the alt attribute in the image tag used for? 
    Used to provide alternate text meaning to an image if it can't be viewed.
    15. How do I reference an id?
    #'id-name'
    16. What is the difference between a section and a div
    A section adds semantic meaning and tells us where sections are broken up on a page.  A div breaks elements to be styled separately.
    17. What is CSS used for?
    CSS is used to style a webpage 
    18. How to we select an element? Example - every h2 on the page
    h2 {}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    A class is less specific than an id, and could be used to apply a style to several elements.  For example you could change the font on several paragraphs with the same class name, and then give the last paragraph an id to change the color of only that paragraph.
    20. How do we select classes in CSS?
    .'class-name'
    21. How do we select a p element with a single class of “human””?
    p.human{}
    22. What is a parent child selector? When would this be useful? 
    A parent child selector is used to select elements that are within another element.  Could be used to select paragraphs within a section.
    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
    A selector that applies to elements in a specific state in the html.

Stretch Questions

    1. What is the difference between an inline element and a block element?
    Inline elements do not break the flow of the page.
    2. What happens when an element is positioned absolutely? 
    It is removed from the document flow, and is positioned to the starting point of its parent element or the page.
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    Using the box model to edit margin and padding, and add a width
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
    Default to block: <p>,<address>,<h1>     Default to inline: <span>,<button>,<abbr>,  Inline-block: <img>
    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
    The box model is used to represent the total spacing of an html element on a page.  The box is broken down into the following parts within one element: content, padding, border, and margin.  All of these parts can be adjusted and styled to determine the total spacing and format of the element.
    The box model fix is box-sizing: border-box.
